## INSTALAÇÃO

* Primeiramento vamos clonar o projeto
```sh 
    git clone https://github.com/gilgledson/vuttr-vue-frontend
```
* acessar a pasta do projeto
```sh 
    cd vuttr-vue-frontend
```
* Iniciar o container
```sh 
    docker-compose up -d --build
```

* acessar o sistema 
```sh 
    http://localhost:8081/
```
